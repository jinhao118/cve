## NETGEAR DGN2200M

Firmware Version: V1.0.0.35_1.0.21NA

Vulnerability Type: Command Injection

Link: https://www.downloads.netgear.com/files/GDC/DGN2200M/DGN2200M%20Firmware%20Version%201.0.0.35%20(North%20America).zip

### Overview

A command injection vulnerability has been discovered in the NETGEAR DGN2200M. The vulnerability lies in the `httpd` module. By sending a message to `ping.cgi`, the `ping_IPAddr` parameter can be manipulated to execute arbitrary commands.
